# Contributing to Anamnexy

Terima kasih atas minat Anda untuk berkontribusi! Karena proyek ini berfokus pada konten medis, mohon ikuti panduan singkat berikut untuk menjaga kualitas dan keselamatan informasi.

1. Fork repositori dan buat branch fitur: `feature/<deskripsi-singkat>` atau fix: `fix/<deskripsi-singkat>`.
2. Setiap tambahan konten medis (obat, dosis, guideline, diagnosis) WAJIB menyertakan minimal satu sumber rujukan resmi (tautan ke pedoman nasional/internasional, jurnal, atau website institusi kesehatan).
3. Sertakan ringkasan singkat perubahan pada deskripsi PR, dan tandai apakah perubahan membutuhkan verifikasi klinis.
4. Gunakan PR template yang tersedia. Reviewer akan memeriksa kelengkapan sumber dan kepatuhan format.
5. Untuk perubahan fungsi (kalkulator dosis), sertakan langkah pengujian manual dan contoh input/output.

Proses review
- PR akan direview oleh maintainer. Untuk konten klinis sensitif, maintainer dapat meminta review oleh rekan medis eksternal.

Kode etik
- Hormati kontributor lain dan ikuti `CODE_OF_CONDUCT.md`.

Terima kasih!
